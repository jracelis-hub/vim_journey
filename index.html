All Things Vim

Introduction

Vim is a highly configurable, powerful and fast text editor. Vim or Vi Improved, is an enchanced version of the older text editor originally called Vi. Vim utilizes the concept of modes for editing. Starting may be daughting but once understanding the fundamentals of how to navigate between modes and muniplating text, the work flow of coding will greatly increase.

Vim is lightweight but there is a whole community that have niches plugins and configurations that can be edited to suit the need of the user.

When using vim the first question that all people ask is how do I get out of this. The default mode upon entering is called Normal mode. To exit it introduces an another mode which is called CMD mode. To enter command mode type the : key. On the bottom of the screen the mouse cursor will be at the bottom, and from there type q to quit. To save edits on a file type :w.

The 3 common command ways to exit are:

  1. :q - quit no edits have been made
  2. :q! - force quit without saving any edits
  3. :wq - save and exit

With the basic of saving and exiting a vim file. In the next section, it will go over how to move the cursor around with key movements.

Navigation

The cursor motions to move around in vim are:

Cursor Movements
Command Description
h to move cursor left
l to move cursor to the right
j to move cursor to the down
k to move cursor to the up
w to move cursor forward to the first character in a word
b to move cursor backward to the first character in a word
e to move cursor forward to the last character in a word
% to move cursor from one matching (,{,[ to the other ),},]
